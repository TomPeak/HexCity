## HexCity 
### Processing P3D
2021 by Tom Peak

Processing patch for a nice endless animation with two functions.

Press c to close or send flying hexagons home.
OSC: /close

Press x to distribute all hexagons randomly on z.
OSC: /xplode

Port: 6666
